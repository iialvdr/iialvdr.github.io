Aplikasi Web To-Do List

JavaScript HTML5 CSS3 Local Storage

Proyek ini adalah aplikasi To-Do List interaktif yang dibangun menggunakan HTML, CSS, dan JavaScript murni (Vanilla JS). Aplikasi ini memungkinkan pengguna untuk mencatat tugas harian, menetapkan tenggat waktu, dan menandai tugas yang telah selesai dengan penyimpanan data yang persisten.

Fitur Utama

CRUD System

Tambah, lihat, tandai, dan hapus tugas dengan mudah.

Local Storage

Data tersimpan otomatis di browser, anti-hilang.

Filter Tugas

Saring tugas berdasarkan status (Selesai/Belum).

Validasi Input

Mencegah data kosong atau tidak valid masuk.
