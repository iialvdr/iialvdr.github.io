Aplikasi Web Jurnal Ibadah

Web App Firebase JavaScript Tailwind

Jurnal Ibadah adalah aplikasi web modern yang dirancang untuk membantu umat Muslim memantau aktivitas ibadah harian. Dengan antarmuka Glassmorphism yang elegan, aplikasi ini dapat diakses secara online kapan saja dan di mana saja.

Fitur Unggulan

Prayer Tracker

Catat dan pantau statistik ibadah sholat harian.

Al-Qur'an Online

Lengkap dengan audio murottal dan terjemahan.

Fitur Lengkap

Asmaul Husna, Tasbih Digital, dan Arah Kiblat.

Sinkronisasi Cloud

Data aman tersimpan dengan Firebase.
